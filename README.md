# Meridian Demo API

Internal REST API for Meridian Engineering. Built with Express and SQLite.

## Quick Start

```bash
npm install
npm run dev
```

## Endpoints

- `GET /health` — Health check
- `GET /users` — List users
- `POST /users` — Create user
- `GET /users/:id` — Get user
- `DELETE /users/:id` — Delete user

## Testing

```bash
npm test
```
